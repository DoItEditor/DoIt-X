## 介绍 <small>Introduce</small>

一个基于Blockly的可视化编辑器，拥有更为现代化的UI和功能，可以使编程像搭积木一样简单。这对很多初学者很有帮助，为青少年提供更方便的编程环境。

<small>A visual editor based on Blockly, with a more modern user interface and functions, makes programming as simple as building with blocks. This is very helpful for many beginners and provides a more convenient programming environment for teenagers.</small>
